The position Property

The position property specifies the type of Positioning method used for an element
There are five different position values:

  • static
  • relative
  • fixed
  • absolute
  • sticky


position: staric;

An element with position: static; is not positiond in nay special way; it is always positioned according to normal flow of the page

position: relative;

An element with position: relative; is positiond relative to its normal position:

The div element has position: relative;

position: fixed;

An element with position: fixed; is positioned relative to the viewport,which means it always stays in the same palce even page is scrolled;

The div element has position: fixed;

position: absolute;

An element with position: absolute; is positioned relative to nearest positioned ancestor (instead of positioned relative to viewport,like fixed):
